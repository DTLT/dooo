  * You may want to embed images in your pages. Please review our [[wordpress:creating-content:uploading-and-displaying-files|tutorial on uploading and displaying media]]. 
  * If you have a collection of photos, you may wish to create an image gallery. You can find . [[http://codex.wordpress.org/The_WordPress_Gallery|basic instructions for creating an Image Gallery in Wordpress here]].

===== Maps =====
  * The simplest way to add a map to your site is to create a My Map in Google Maps. You will need a Google account in order to create one. 
    * [[https://www.google.com/maps/d/|You can create a Google Map here.]] 
    * You can find [[https://support.google.com/maps/answer/3045850?hl=en|a tutorial about how to create a Google Map here.]
    * You can find [[https://support.google.com/maps/answer/3544418?hl=en|a tutorial about how to embed a Google Map here.]] Important Note: You will need to paste the embed code in WordPress in the Text editor (Read more at [[wordpress:creating-content:visual-editor-vs-text-editor|Visual Editor vs. Text Editor]]
