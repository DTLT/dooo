===== Maps =====
  * The simplest way to add a map to your site is to create a My Map in Google Maps. You will need a Google account in order to create one. 
    * [[https://www.google.com/maps/d/|You can create a Google Map here.]] 
    * You can find [[https://support.google.com/maps/answer/3045850?hl=en|a tutorial about how to create a Google Map here.]]
    * You can find [[https://support.google.com/maps/answer/3544418?hl=en|a tutorial about how to embed a Google Map here.]] Important Note: You will need to paste the embed code in WordPress in the Text editor (Read more at [[wordpress:creating-content:visual-editor-vs-text-editor|Visual Editor vs. Text Editor]]
  * If you're looking for a more advanced Map tool, we recommend you check out the [[https://www.arcgis.com/home/signin.html?returnUrl=http%3A%2F%2Fwww.arcgis.com%2Fapps%2FMapTour%2F%3FfromScratch | Map Tour Creator]] provided (for free) by ESRI. Map Tour allows you to create a map-based story by plotting a sequential, place-based tour of an area. You can use it to geo-tag a series of images and captions that are then linked to an interactive map. Please note that you cannot embed a MapTour in WordPress -- however, you can link to your MapTour from a page in WordPress.
    * In order to use it, you will need to have a [[https://www.arcgis.com/home/signin.html|free, public account with ESRI]].
    * You can [[http://storymaps.arcgis.com/en/app-list/map-tour/ | learn more about the Map Tour app here]].
    * You can [[http://storymaps.arcgis.com/en/app-list/map-tour/tutorial/ | find detailed step-by-step instructions here.]]
    * You can [[http://storymaps.arcgis.com/en/app-list/map-tour/gallery/#s=0&n=30&d=1&md=storymaps-apps:000100000 | find examples of Map Tour stories here]].
