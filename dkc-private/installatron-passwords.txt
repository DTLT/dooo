====== Change Password for Applications using Installatron ======

Once logged into the appropriate user account for Domain of One's Own, you can now change the password for any application using Installatron.

1. Scroll down to "**Web Applications**."
{{https://farm4.staticflickr.com/3900/15394291165_1656a545a2_b.jpg}}
2. Click on the desired application.
3. Scroll down to where you see "**Administrator Username**" and "**Administrator Password**."
{{https://farm4.staticflickr.com/3924/15207619860_9e0e9110cc_b.jpg}}
4. Enter in a new password (or username if desired).
5. Scroll down and press "**Save All**."
