====== Adding Zotero ======
Zotero is a research tool that automatically senses content in your web browser and then allows you to add it to your personal library with a single click. Zotero allows you to organize research material and generate citations in both a note and bibliography form. To get Zotero go to [[https://www.zotero.org/]].
There are two ways to add Zotero. 
  - Zotero for Firefox
  - Zotero Standalone

==== Zotero for Firefox ====
Zotero for Firefox lets you capture and organize all your research without ever leaving the browser.
=== Adding Zotero to Firefox ===
  - **To add Zotero directly to Firefox**, go to [[https://www.zotero.org/download/]] using the Firefox Browser and click on the Zotero 4.0 for Firefox button.{{https://farm4.staticflickr.com/3933/15578338376_d9636d2172_c.jpg}}
  - Click **Allow** when Firefox prompts you with a security pop-up. {{https://farm6.staticflickr.com/5599/15416019507_f2be917df5_o.png}}
  - Click **Install** in the following pop-up. {{https://farm6.staticflickr.com/5616/15415904638_a1d076325f_o.png}}
  - Click **Restart** for Zotero to be on Firefox. 
  - Zotero has now been added to Firefox! You can **register** to be able to access your library on any computer with Firefox. {{https://farm6.staticflickr.com/5609/15415904508_a02c40c015_c.jpg}}

To learn how to use Zotero, [[dkc-private:using-zotero|go here]]. 
To learn more about Zotero, you can go to [[https://www.zotero.org/support/quick_start_guide]]. 


====  Zotero Standalone ====
 Zotero Standalone runs as a separate application and plugs into your choice of browser. It works with Chrome, Safari, and Firefox. 

=== Installing the Standalone ===
 
1. Go to https://www.zotero.org/ and select **Download Now**
{{https://farm4.staticflickr.com/3953/15416359487_80a72c3c45_b.jpg}}

2. Choose **Zotero Standalone** and select your preferred browser. 
{{https://farm6.staticflickr.com/5602/15599692261_ca09f22072_o.png}}

3. When downloading the Zotero Standalone you will be asked to give permission for it to run. Select **Open**.
{{https://farm6.staticflickr.com/5600/15603191912_a64766755e_o.png}}

4. You have successfully downloaded Zotero! Now register your account!
{{https://farm6.staticflickr.com/5606/14982283553_fd69c884b5_b.jpg}}

=== Chrome App===
If you are using Chrome as your web browser you will need to also download the Zotero App for Chrome. 

1. Log into your Google Chrome Account

2. Navigate to https://chrome.google.com/webstore/category/apps 

3. Search for **Zotero**

{{https://farm6.staticflickr.com/5601/15603426982_3915da7288_o.png}}

4. Install the App
{{https://farm4.staticflickr.com/3952/15578963416_454109ab30_b.jpg}}

5. Now when on an article or other web source you will see a new icon in your browsers URL
{{https://farm6.staticflickr.com/5605/15416134439_501f8a9c75_o.png}}

You will now have the ability to sync your Zotero application with the Chrome web browser. 

=== Getting Started ===

Zotero is, at the most basic level, a citation manager. It is designed to store, manage, and cite bibliographic references, such as books and articles. In Zotero, each of these references constitutes an item.
