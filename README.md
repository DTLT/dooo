#Domain of One's Own Documentation



[A Domain of One's Own](http://umwdomains.com) is a project from the [Division of Teaching and Learning Technologies](http://academics.umw.edu/dtlt) at the [University of Mary Washington](http://umw.edu). This repository is a mirrored archive of the wiki documentation available at [http://docs.umwdomains.com](http://docs.umwdomains.com). We welcome contributions by forking the repository and creating pull requests.
