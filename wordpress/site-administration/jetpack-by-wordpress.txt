====== Jetpack ======
  * [[https://wordpress.org/plugins/jetpack/|Jetpack]] is a plugin for WordPress that adds several useful tools including customization, traffic, mobile, content, and performance tools.
  * To install Jetpack to your website, log on to your WordPress dashboard and make sure you have the plugin [[wordpress:site-administration:managing-comment-spam-with-akismet|Akismet]] installed and activated.
  * Now it's time to install Jetpack. From your dashboard, click on "**Plugins**" in the left-hand menu. Then choose “**Add New**.” 
{{https://farm8.staticflickr.com/7558/16108870457_4b8f509701_c.jpg}} 
  * You’ll then see a list of featured plugins. You’ll want to click install now beside “**Jetpack by WordPress.com**.” If you do not see Jetpack, search for it in the right-hand menubar
{{https://farm8.staticflickr.com/7483/15674850103_92672d58aa_c.jpg}}
  * Once Jetpack is successfully installed, be sure to click on "**Activate Plugin**". Once you activate the plugin, you then be prompted to “**Connect to WordPress.com**.”
{{https://farm8.staticflickr.com/7504/15672622064_9167e8b032_c.jpg}} 
  * Now you will be prompted to log into [[https://wordpress.com/|WordPress.com]]. If you do not have an account, click on "**Need an account?**" to sign up. 
{{https://farm9.staticflickr.com/8584/15675159023_2e258f8193_z.jpg}}
  * Afterwards, click “**Authorize Jetpack**” and you’re ready to use the plugin.
{{https://farm9.staticflickr.com/8586/15675175253_0430c935bc_z.jpg}}
