====== Installing MediaWiki ======


<html>

<p>MediaWiki is an open-source publishing platform that can be used for creating a collaborative document repository. It's the software that drives the <a href="http://wikipedia.com">Wikipedia</a> website. Setting up a MediaWiki install on your own domain can be done by following these steps:</p>
<ol>
<li>To get started you'll need to login to your control panel by going to the UMW Domains homepage and clicking&nbsp;<strong>Login</strong>.<br /><br /><img style="border: 1px solid black;" src="http://farm4.staticflickr.com/3808/9394644932_42072c3f76_c.jpg" alt="UMW Domains Login" width="800" height="409" /><br /><br /></li>
<li>Here you'll login with the username and password that was provided to you with the Welcome email during the signup process.<br /><br /><img src="http://farm4.staticflickr.com/3834/9394646464_cb1e711493_c.jpg" alt="UMW Domains Login Page" width="800" height="709" /><br /><br /></li>
<li>Once logged in you'll be at the homepage of your control panel.The easiest way to navigate the panel is using the search feature in the top right panel. Let's use it now to search for MediaWiki.<br /><br /><img src="http://farm6.staticflickr.com/5521/9394646810_9b824be97d_c.jpg" alt="UMW Domains Control Panel Home" width="789" height="800" /><br /><br /></li>
<li><span>As you can see when you search a term, the corresponding section is shown. Click the <strong>MediaWiki</strong>&nbsp;</span><span>button.<br /><br /><img src="http://farm8.staticflickr.com/7293/9411304686_241be5d546_c_d.jpg" alt="cPanel MediaWiki" width="800" height="610" /><br /><br /></span></li>
<li><span><span>The next page gives you more information about the MediaWiki software. To begin the install, click </span><strong>install this application</strong><span>&nbsp;in the upper-righthand corner.<br /><br /><img src="http://farm8.staticflickr.com/7389/9411304584_10155ebb75_c_d.jpg" alt="MediaWiki Installatron Info Page" width="800" height="582" /><br /><br /><br /></span></span></li>
<li><span><span><span>On the next page the installer will ask for some information about this install. The first thing you'll want to do is decide where to install it. If you're wanting to install MediaWiki on your main (the root) domain, you can leave the directory area empty. If you created a subdomain, you can select it from the dropdown menu. You also have the option of installing MediaWiki in a subfolder by typing in the folder name in the&nbsp;</span><strong>Directory</strong><span>&nbsp;field. In this instance, we have opted to install it in a directory called "mediawiki". Click here for more information about subdomains and subfolders.<br /><br /><img src="http://farm6.staticflickr.com/5327/9408541979_770aca41eb_c_d.jpg" alt="MediaWiki Installatron Location" width="800" height="735" /><br /><br /><br /></span></span></span></li>
<li><span>By default the installer will automatically backup your MediaWiki website and update it anytime a new version comes out. While we recommend you keep this option, it is possible to only do minor updates, or turn them off completely. The installer will also create a database for you automatically, but if you've already created one for this website you can choose&nbsp;</span><strong>Let me manage the database settings</strong><span>&nbsp;and enter the details.<br /><br /><br /></span></li>
<li><span><span>Finally, you'll need to create an initial username and password for the WordPress install. Enter that information in final section and click&nbsp;</span><strong>Install</strong><span>. (note: A default username and password are provided for you)<br /><br /><img src="http://farm4.staticflickr.com/3770/9411304662_24ec5dee97_c_d.jpg" alt="MediaWiki Installatron Settings" width="800" height="711" /><br /><br /></span></span></li>
<li><span>The installer will take just a few moments to install WordPress and a progress bar will keep you updated. When it is complete you will see a link to your new MediaWiki site as well as a link to the backend administrative section for your MediaWiki site.<br /><br /><img src="http://farm4.staticflickr.com/3833/9411304644_bcd7cc5519_c_d.jpg" alt="MediaWiki Installatron My Applications" width="800" height="151" /><br /><br /><br /></span></li>
<li><span><span>Congratulations, you have now completed the installation of MediaWiki! You can now start to create collaborative documents on your own domain.<br /><br /><img src="http://farm3.staticflickr.com/2881/9408541867_6f2ac2ed46_c_d.jpg" alt="MediaWiki Screen" width="800" height="436" /><br /></span></span></li>
</ol><br /><br />

</html>
